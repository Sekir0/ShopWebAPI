# WebAPI

## Simple Web Api
## На данный момент в приложении реализованы следующие функции:
- CRUD для продуктов
- CRUD для категорий
- JWT авторизация 
- Identity регистрация
- Все под automapper'oм
- Paginator
- Redis простенькие кеши
- Healthcheck - для проекта вцелом
